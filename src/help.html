New playback speed option

Playback speed cannot work on pages where videos are actually hosted on a different site. (e.g. youtube videos not on the youtube site). This is a limitation of the Chrome Extension API itself.

When a site looks like it supports speed control, the toolbar icon will display a overlay
When the speed control graphics is displayed, clicking the button shows extra options to control playback speed.

Why speed control fails on some sites

When any pages loads their videos from a different site (e.g. a youtube video embedded in a non youtube site), then the speed control feature will not work and is disabled. Video Maximizer will fall back to the classic toggle behavior.

When I site doesn't support zooming, then a overlay will be displayed.

New keyboard shortcuts

  1. The Unzoom function can be accessed by clicking this icon (or the Spacebar after opening)
  2. Clicking the speed-change button a second time toggles it back to normal speed.
  • up/down keys: Increase and decrease playback speed. See 1
  • right/left keys: Skip the playback forwards/back. See 2
  • arrow + shift: Longer skips.
  • relative skip times: Changing playback speed adjusts how long the skip is.
    0.25x playback speed is 1/4th less the skip time.
    e.g. 10s skip at 16x playback would skip forward 1m 40s in the video.
  • Length of skips Can be customized in the Options page.
  • TIP: ‚≠ê Pressing the Escape key on the page UNZOOMS.
  • TIP: Pressing escape when the Speed Control window is open will close it.
  • NOTE: Speed changing will not work on sites hosting videos from other sites. This is just a limitiation of the Chrome APIs.
  • NOTE: there is an option to disable the Speed Controls to revert to the behavior the original Video Maximizer (see Options below)
  • Keyboard shortcut to zoom: This is built into chrome! Go to chrome://extensions/shortcuts in Chrome

Pinning the extension to the toolbar makes it easier to use.

  1. To pin, first click on the Extensions puzzle piece icon
  2. Click the Pin icon

Changing Options

  1. Right-click the icon
  2. Select Option from the dropdown menu.
You can disable the speed controls and revert the behavior back to the classic toggle.
