Chrome Update Bug Warning

There is currently a bug with Chrome that impacts ALL extensions. Right after an update, extensions can just stop working. There are two user workaround.

The first is to restart Chrome.

The second is to go to Manage Extensions under the puzzle piece in the Chrome toolbar. Then enable Developer mode and manually reload the extension.


Hopefully this will be fixed soon and only seems to show up after the extension is modified. See Issue #1316588 for more information. But this trick is handy for all extensions until it's resolved.

New playback speed option

Playback speed cannot work on pages where videos are actually hosted on a different site. (e.g. youtube videos not on the youtube site). This is a limitation of the Chrome Extension API itself.

When a site looks like it supports speed control, the toolbar icon will display a overlay
When the speed control graphics is displayed, clicking the button shows extra options to control playback speed.

Why speed control fails on some sites

When pages load their videos from a different site (e.g. a youtube video embedded in a non youtube site), then the speed control feature often will not work.

If this happens, there's a setting to try. "Allow access to all sites by default" in the Options will increase this extension's permissions.

The fundamental issue is that cross-domain-frame-permissions-granting doesn't work in Chrome yet. See Issue #826433

New keyboard shortcuts

  1. The Unzoom function can be accessed by clicking this icon (or the Spacebar after opening)
  2. Clicking the speed-change button a second time toggles it back to normal speed.
  • right/left keys: ⬌ Skip the playback forwards/backwards.
  • up/down keys: ⬍ Increase and decrease playback speed.
  • arrow + shift: Longer skips.
  • relative skip times: Changing playback speed adjusts how long the skip is.
    0.25x playback speed is 1/4th less the skip time.
    e.g. 10s skip at 16x playback would skip forward 1m 40s in the video.
  • Length of skips Can be customized in the Options.
  • TIP: Pressing the Escape key on the page UNZOOMS.
  • TIP: Pressing escape when the Speed Control window is open will close it.
  • NOTE: Speed changing will not work on sites hosting videos from other sites without granting this extension more permissions in the Options. This is just a limitation of the Chrome APIs.
  • NOTE: there is an option to disable the Speed Controls to revert to the behavior of the original Video Maximizer (see Options below)
  • Keyboard shortcut to zoom: This is built into chrome! Go to chrome://extensions/shortcuts

Known issues

Youtube controls are not fullscreen

The solution for this is to put the youtube video into "theater" mode before using Video Maximizer. This can be done by pressing the "t" key.


Skipping doesn't work on Netflix

Skipping the video backwards on netflix causes an error. There's no workaround, so this feature is just disabled for now.


Speed controls don't work on some sites

Video Maximizer attempts to use the minimal permissions required to do it's job. However, there are still some sites that require full access to every domain to work. (technical reason has to do with embedding iframes that have cross-domain references).

The only current work around is to enable "Allow access to all sites by default" in Video Maximizer's Options.

There's an open Chrome issue/feature request to allow this to work without needing this extreme access.


Disabling "Allow access to all sites by default" doesn't reset permissions

Another Chrome issue. Video Maximizer removes the extra permissions; however, since they've already been granted, it seems like the extension must be uninstalled and reinstalled to reset it.


Constant security prompting

You should only get prompted ONCE per unique site you use Video Maximizer on; however, if this annoys you, then you have the choice to enable the "Allow access to all sites by default" option. You will then get one last prompt and never be bugged again.

Changing Options

  1. Right-click the icon
  2. Select Option from the dropdown menu.
You can disable the speed controls and revert the behavior back to the classic toggle.
