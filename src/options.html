Allow full access to all sites by default
WARNING: It will allow FULL ACCESS to ALL SITES by default.
This access should not be required, but is a fix-all. Once granted and accepted, it silences all further permissions prompting.

If you find a site that still doesn't work after enabling this setting, then consider reporting it via a GitHub issue ↗.
Playback skipping
While Video Maximizer popup controls are displayed, you can use the arrow keys to skip playback.  Playback skipping is in viewing times, not video time. For example, 2x playback speed skips twice as far. Read more on the help page
