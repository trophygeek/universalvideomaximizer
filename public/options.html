Playback skipping
While Video Maximizer popup controls are displayed, you can use the arrow keys to skip playback.  Playback skipping is in viewing times, not video time. For example, 2x playback speed skips twice as far. Read more on the help page
(when Shift key is used)
Activation Hotkey
To have a hotkey activate VideoMaximizer on page, simply use the built-in Chrome feature.
Go to Extensions in Chrome and select "Keyboard shortcuts" from the left.
Or navigate directly to chrome://extensions/shortcuts
